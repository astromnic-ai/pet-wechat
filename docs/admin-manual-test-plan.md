# 后端管理功能人工测试清单

> 依据 PRD v0.8《宠物在场 - 桌面摆台》第六章"后端管理功能" 2.1–2.7 整理。
> 用于管理后台（`https://pet-admin.yangl.com.cn`）上线前的人工回归测试。
> 每条轨迹都可独立执行，完成后在前面的 `[ ]` 打勾。

## 2.1 系统概览（Dashboard）

- [ ] **轨迹 1-1 · 数据汇总准确性** [P0]
  打开概览页 → 核对"总用户数 / 活跃设备 / 今日互动总数"。
  验证点：数字能加载；与数据库/其它页面一致；今日互动是否实时更新。

- [ ] **轨迹 1-2 · 设备分布占比** [P1]
  查看"设备类型分布"图 → 桌面端 vs 项圈数量占比显示是否正确（可在"设备管理"里交叉核对）。

- [ ] **轨迹 1-3 · 设备活跃度分档** [P1]
  查看活跃度统计 → 验证高/中/低分档（按 3 天周期：高 >30×3、中 6×3–30×3、低 <18）与实际互动数吻合。

- [ ] **轨迹 1-4 · 实时动态面板** [P1]
  查看"新设备上线 / 长时间离线 / 新增审核 / 个性化定制 / 电量预警"条目 → 触发一次对应事件（例如新增一台设备），等待刷新周期后确认列表更新。

- [ ] **轨迹 1-5 · 系统健康状态** [P2]
  查看"服务器/API/数据库/云存储"状态（当前为占位，确认不报错即可）。

## 2.2 宠物行为日程管理

- [ ] **轨迹 2-1 · 切换宠物品类** [P0]
  顶部下拉切换"猫 / 狗 / 其他" → 日程表随之切换；每个品类独立保存。

- [ ] **轨迹 2-2 · 拖拽添加基础动作** [P0]
  从动作列表把"蹲坐/吃饭/睡觉/趴卧/跑/走"其一拖到时间轴 → 时间块能放下、能拉伸缩小。

- [ ] **轨迹 2-3 · 拖拽趣味动作** [P0]
  把"玩球/噗噗/看电视/追尾巴/挠空气/做美梦/舔爪子/wink转圈"之一穿插加入 → 正常显示。

- [ ] **轨迹 2-4 · 时间段变更与重叠校验** [P0]
  拉伸/移动时间块 → 确认是否支持重叠，以及跨 0 点边界的表现。

- [ ] **轨迹 2-5 · 生效策略选择** [P0]
  切换"每天 / 工作日 / 节假日" → 保存后重新进入页面确认已持久化。

- [ ] **轨迹 2-6 · 保存并应用** [P0]
  点击"保存配置"→ 选择应用模式 → 在设备侧（如有联调）验证配置下发生效。

## 2.3 图像审核中心

- [ ] **轨迹 3-1 · 顶部任务计数** [P0]
  核对"今日待处理 / 今日已完成 / 同步设备总数"数字；新增一张待审核后数字 +1。

- [ ] **轨迹 3-2 · 分类 Tab 切换** [P0]
  切换"待审核 / 已通过 / 已拒绝" → 缩略图列表正确刷新；按最新时间排序。

- [ ] **轨迹 3-3 · 缩略图信息展示** [P0]
  每张图显示：宠物信息、上传时间、审核状态。

- [ ] **轨迹 3-4 · 审核通过流程** [P0]
  点进详情 → 点"通过" → ①图像进入"定制中心"的待定制池；②小程序端收到"图像审核通过，进入定制"消息推送；③审核框状态更新为"已通过"。

- [ ] **轨迹 3-5 · 审核拒绝流程** [P0]
  点"拒绝" → 下拉选通用原因（如"图像有遮挡 / 面部不完整"）→ 保存 → 小程序端收到拒绝推送；也验证"手动输入原因"路径。

- [ ] **轨迹 3-6 · 下载原图** [P0]
  在详情页点下载 → 本地拿到原图，文件名/格式无误。

## 2.4 定制中心

- [ ] **轨迹 4-1 · 任务标签计数** [P0]
  核对"累计定制总量 / 今日新增"数字，审核通过一条后数字变化。

- [ ] **轨迹 4-2 · 卡片列表筛选** [P0]
  按"基本图像 / 个性化定制"筛选 → 列表切换正确；卡片显示"原图、微信昵称、用户ID、宠物信息、分类"。

- [ ] **轨迹 4-3 · 打开动作定制池** [P0]
  点一张卡片 → 右侧显示该宠物的 24 个动作占位（基础 + 主动交互），进度条/完成数正确。

- [ ] **轨迹 4-4 · 上传定制素材** [P0]
  点"+"上传本地视频 → 显示缩略图，状态由空变"已上传"（✔）。

- [ ] **轨迹 4-5 · 动作状态流转** [P0]
  未上传 → 部分完成（"进行中"）→ 全部完成（"已完成"），列表标签对应变化。

- [ ] **轨迹 4-6 · 填写宠物描述** [P0]
  在定制详情里手动填写"宠物描述/趣味点"→ 保存后复查内容。

- [ ] **轨迹 4-7 · 一键同步用户端** [P0]
  完成全部 24 个定制后点"同步用户端" → 小程序端对应宠物收到全部素材与描述，消息中心有通知。
